How the Web Works

Your browser makes requests via the Internet to servers. These requests are use a protocol called HTTP. And the servers respond with files that the browser displays. Some examples of browsers include Chrome, Safari, Firefox, Internet Explorer, and Edge. You're probably running one of those right this moment. These servers are computers just like yours except they are optimized for sitting in a closet and hosting files rather than sitting on your desk browsing these files. And the protocol HTTP is just a very simple protocol that your browser uses to communicate with these servers. About HTML

HTML stands for HyperText Markup Language, and it is the most widely used language to write Web Pages.

  • Hypertext refers to the way in which Web pages (HTML documents) are linked together. Thus, the link available on a webpage is called Hypertext.
  • As its name suggests, HTML is a Markup Language which means you use HTML to simply "mark-up" a text document with tags that tell a Web browser how to structure it to display.

Tags and Elements

HTML documents are made up of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. An HTML element is defined by a starting tag. If the element contains other content, it ends with a closing tag, where the element name is preceded by a forward slash. This video explains the distinction well

Attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

Inline vs Block Elements

  • A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
  • An inline element does not start on a new line and only takes up as much width as necessary
  • Some of the inline elements are <b>, <em>, <img>, <span> etc.
  • Example of block elements are <div>, <p>, <form> etc.

Blank Spaces

HTML assumes multiple white spaces as a single space. In order to give spaces, we can use the following:

  • <br> for line breaks
  • &nbsp; for a single space
  • <p> for a new paragraph (leaves some space after previous block of text)

What is CSS

CSS refers to Cascading Style Sheets. These are use to add styles to webpage There can be more than one css in the website and the most local declaration is given maximum priority.

HTML-CSS DOM

  • HTML has a tree like structure, wth elements nested inside other elements. Link to Udacity video. When programmers talk about the DOM(Document Object Model) they are talking about the tree like structure of a web page.
  • HTML controls the "structure" of the web page
  • CSS controls the "style" of a page (How it looks)
  • CSS helps web designers avoid repetition. Avoiding repetition helps to keep code "clean" and more easily modifiable.
  • Boxes- webpages are composed of "boxes". <div> tag defines boxes. Class attributes are the labels on boxes.

3 Ways to add CSS

  • Inline CSS - defined within the html tag
  • Internal CSS - difined within the html file
  • External CSS - defined out side html file

The Box Model

Everything on a webpage is a box. Every box has the following:

  • Margin - Space outside the border, which determines how far is it from adjacent boxes
  • Border - Line which marks the boundry of a box
  • Padding - Space between border and content
